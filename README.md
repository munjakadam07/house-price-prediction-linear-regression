# House Price Prediction using Linear Regression

## Project Overview
This project uses Supervised Machine Learning to predict house prices using Linear Regression.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Workflow
1. Data preprocessing
2. Feature engineering
3. Feature scaling
4. Model training
5. Model evaluation

## Results
The Linear Regression model was trained to estimate house prices based on property features and achieved reliable prediction accuracy.
